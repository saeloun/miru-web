# README

## Miru-Web

Saeloun timetracking application.

### Installation

Step 1: Clone repo to local
```
git clone https://github.com/saeloun/miru-web.git
```
Step 2: Install gem 
```
bundle install
```
Step 3: Install node packages
```
yarn install
```
Step 4: Running app in local
```
foreman start -f Procfile.dev
```
